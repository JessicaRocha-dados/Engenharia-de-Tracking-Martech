# Case Study: Refatoração de Tracking de Formulário

## 📌 Visão Geral do Projeto
Este projeto documenta a evolução da arquitetura de rastreamento de um formulário de captação de leads. O objetivo foi migrar de um acionador genérico para uma estrutura robusta utilizando **[Data Layer / Element Visibility]** e **JavaScript Personalizado**, garantindo a integridade dos dados enviados para o Google Analytics 4 e Meta Ads.
